■ 背景画像の配置場所

各セクションに対応する画像ファイルをこのフォルダに入れてください。
推奨フォーマット：JPG / WebP
推奨サイズ：横1920px 以上

ファイル名（変更不可）:
  hero.jpg          → トップ（ヒーロー）セクション
  profile-bg.jpg    → プロフィールセクション
  career-bg.jpg     → 経歴セクション
  stats-bg.jpg      → 成績セクション
  gallery-bg.jpg    → ギャラリーセクション
  news-bg.jpg       → ニュースセクション
  message-bg.jpg    → 応援メッセージセクション

※ 画像がない場合はグラデーション背景にフォールバックします。
